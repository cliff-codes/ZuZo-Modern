# Partner Portal Component Structure

This document outlines the component breakdown for the Partner Portal page.

## Component Hierarchy

```
partner-portal.tsx (Main Page)
├── PartnerNavigation
├── PartnerHero
│   └── PartnerNetworkVisualization
├── PartnerTracksSection
│   └── PartnerTrackCard (x4)
├── MarginCalculator
│   ├── CalculatorInputs
│   └── CalculatorResults
├── TechnologyStackSection
│   └── TechnologyTabs
├── ResourcesSection
│   └── ResourceCard (x3)
├── TestimonialsSection
├── FAQSection
├── CertificationSection
├── BookingSection
└── DownloadModal
```

## Components to Create

1. ✅ PartnerNavigation - Navigation bar with mobile menu
2. ✅ PartnerHero - Hero section with headline and CTA
3. ⏳ PartnerNetworkVisualization - Animated network graph
4. ⏳ PartnerTracksSection - The 4 partner track cards (MSP, Master Agent, NDIS, 3PL)
5. ⏳ MarginCalculator - Interactive ROI calculator
6. ⏳ TechnologyStackSection - Technology integration tabs
7. ⏳ ResourcesSection - Gated resource library
8. ⏳ TestimonialsSection - Partner testimonials
9. ⏳ FAQSection - Frequently asked questions
10. ⏳ CertificationSection - Partner certification program
11. ⏳ BookingSection - Cal.com booking integration
12. ⏳ DownloadModal - Email gate for resource downloads
